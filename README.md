# Online Courses Recommendation System

Интеллектуальная система рекомендаций онлайн-курсов, ориентированная не на рейтинги и маркетинг, а на **реальные карьерные результаты выпускников** и персональные цели пользователя.

---

## Проблема

Люди выбирают онлайн-курсы «вслепую»:
- ориентируются на звёздочки и описания,
- не понимают, подойдёт ли курс именно им,
- не могут оценить реальный карьерный эффект до начала обучения.

---

## Решение (подход проекта)

Проект решает эту проблему за счёт:

- анализа **карьерных траекторий выпускников** вместо субъективных рейтингов;
- построения **персонального профиля пользователя** через интерактивный диалог;
- **ML-ранжирования курсов** (XGBoost) по вероятности достижения цели;
- прозрачных **объяснений рекомендаций** на основе данных и примеров выпускников.

---

## Что делает продукт

Система:
1. Собирает и анализирует синтетические данные о курсах, навыках и карьерных исходах.
2. Ведёт интерактивный диалог с пользователем, извлекая цели и ограничения из свободного текста.
3. Ранжирует курсы под конкретного пользователя с помощью Learning-to-Rank.
4. Объясняет, **почему именно эти курсы рекомендуются**, используя метрики и примеры выпускников.

---

## Архитектура и этапы

1. **Подготовка данных** — генерация и очистка синтетических датасетов.
2. **NLP-анализ** — извлечение навыков, ролей и сигналов успеха (spaCy, SkillNER).
3. **Оценка исходов** — расчёт времени до работы, смены домена, роста зарплаты.
4. **Агрегация курсов** — объективные метрики эффективности курсов.
5. **Построение признаков** — адаптация курсов под профиль пользователя.
6. **ML-ранжирование** — XGBoost Learning-to-Rank.
7. **Интерактивный диалог и объяснения** — LLM через Ollama.

---

## Технологии

- Python
- pandas, numpy
- spaCy (en_core_web_md, ru_core_news_md)
- SkillNER
- XGBoost (Learning-to-Rank)
- Ollama (LLM для диалога и объяснений)

---

## Как установить

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_md
python -m spacy download ru_core_news_md
```

Для работы требуется установленный **Ollama** и локальная LLM.

---

## Как запустить (пример)

```bash
python main.py
```

Система:
- начнёт диалог с пользователем,
- уточнит цель и ограничения,
- покажет топ курсов,
- объяснит каждую рекомендацию.
